🚀 Codomax Internship | Day 2 of 15

Today I built a mini blog website from scratch using HTML, CSS, and JavaScript!

Here's what I worked on:
🏠 Home Page — a clean landing page with a hero section and a dynamic feed of blog entries
✍️ Add Blog Page — a form to create and publish new posts on the fly
🧭 Navigation Bar — responsive, with a mobile menu toggle
📌 Footer — consistent across pages with quick links and categories

A few things I focused on:
✅ Designing a cohesive visual identity instead of a generic template
✅ Using localStorage so new posts persist in the browser
✅ Making the layout fully responsive across devices
✅ Keeping the codebase modular — shared CSS & JS across pages

It's a small project, but it pushed me to think beyond just "making it work" — into how design, structure, and user experience come together.

Excited for what Day 3 has in store! 💻
